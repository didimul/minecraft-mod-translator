1 инструкция:
    Если нужен перевод с анлийского на русский:
        - Закинуть файл en_us.json (открыть .jar файл мода, я обычно открываю через 7z, зайти в assets =) в папку input
            • Откуда взять en_us.json:
                1: Открыть .jar файл мода (я обычно открываю через 7z)
                2: Зайти в assets => "название мода" => lang
                3: Взять en_us.json
        - Открыть файл start.bat
        - Дождаться завершения перевода (перевод через Google Translate)
        - Открыть папку output
        - Забрать файл ru_ru.json
        - Открыть файл мода (.jar файл)
        - Зайти в assets => "название мода"  => lang
        - Закинуть файл ru_ru.json в папку lang

2 инструкция:
    Если нужен перевод с другого языка на другой:
        - Открыть файл language.json
        - Где написано "fromLang":
            1: Убрать "en"
            2: Написать в двойных кавычках "язык с которого делается перевод" 
            !!! В кратком (ISO 639) формате, например, english = en или russian = ru

        -  Где написано "toLang":
            1: Убрать "ru"
            2: Написать в двойных кавычках "язык на которого надо сделать перевод"
            !!! Тоже в кратком (ISO 639) формате и обязательно в двойных кавычках

        - Где написано "inputFile":
            1: Убрать "en_us.json"
            2: Написать язык в кратком формате, но чуть по другому (пример: ru (Russian) => ru_ru.json или de (Deutsch) => de_de.json)

        - Где написано "outputFile":
            1: Убрать "ru_ru.json"
            2: Написать язык в кратком формате, но чуть по другому (пример: en (English (US)) => en_us.json или de (Deutsch) => de_de.json)

        - Дальше следовать по 1 инструкции

Сокращенный формат языка для "inputFile" и "outputFile" можно посмотреть тут https://minecraft.fandom.com/wiki/Language (возможно и для "fromLang" и "toLang", я не проверял)
